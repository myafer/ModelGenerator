# ModelGenerator
Mac App that help you to convert JSON to Mappable model.

# Screenshot 
![GitHub API](https://github.com/sheepy1/ModelGenerator/raw/master/screenshot.png)

# Usage
- Open the ModelGenerator in MacApp directory.
- Input model name and JSON.
- Copy and paste.
